Какие технолигии использовались к данной курсовой работе

Spring Data – дополнительный удобный механизм для взаимодействия с сущностями базы данных, организации их в репозитории, извлечение данных, изменение и т.д.
Spring Security - это Java/Java EE фреймворк, предоставляющий механизмы построения системы аутентификации и авторизации, а также другие возможности обеспечения безопасности для промышленной разработки приложения;
Hibernate — библиотека для языка программирования Java, предназначенная для решения задач объектно-реляционного отображения, самая популярная реализация спецификации JPA. Распространяется свободно на условиях GNU Lesser General Public License.
Spring MVC – это универсальный фреймворк с открытым исходным кодом для Java-платформы.
Spring REST – это архитектурный стиль взаимодействия приложения в сети. Данный фреймворк предоставляет богатый набор инструментов упрощающий разработку REST API: инструменты для маршрутизации запросов, классы-кодеки для преобразования JSON/XML в объекты требуемых типов.
Spring Framework — универсальный фреймворк с открытым исходным кодом для Java-платформы. Также существует форк для платформы .NET Framework.
